# Speekify Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Python TUI that converts pasted French text or readable content extracted from a pasted URL into a correctly named `.wav` file under `output/`.

**Architecture:** The app is split into a Textual TUI, extraction helpers for pasted text and URLs, deterministic output naming, and a thin Supertonic adapter. The UI orchestrates validation, extraction, naming, synthesis, and user-visible status updates without embedding synthesis logic in widgets.

**Tech Stack:** Python 3.11+, UV, Textual, Supertonic Python package, httpx, trafilatura, pytest.
